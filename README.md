# SigNote 官方網站

SigNote 官網（行銷＋隱私／條款／客服）的**公開**靜態站。純 HTML/CSS，無建置步驟。

- 線上網址：https://chshlingit.github.io/signote-site/
- 推到 `main` 會由 GitHub Actions 自動部署（`.github/workflows/deploy.yml`）。

> 此 repo 只放官網（公開內容）；App 原始碼在私有 repo `signote`。
> 官網開發來源為 `signote/site/`，更新時由該 repo 的同步腳本推送過來。

## 內容
- `index.html` — 行銷首頁（Marketing URL）
- `privacy.html` — 隱私權政策（App Store 必填）
- `terms.html` — 使用條款＋社群守則
- `support.html` — 客服 FAQ（App Store 必填）
- `style.css`、`assets/` — 樣式與示範圖
